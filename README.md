# Retail Sales MIS Dashboard — Fashion Wave
### Excel | Power Query | PivotTables | Data Visualization

## Business Problem
A fashion retail brand needed visibility into sales performance, 
store profitability, and pricing effectiveness across multiple 
locations. Manual reporting was slow and inconsistent.

## Objective
Build an end-to-end MIS reporting dashboard to identify revenue 
trends, discount impact, and profitability opportunities.

## Tools Used
- Microsoft Excel — PivotTables, Advanced Formulas, Slicers
- Power Query — Data Cleaning and Transformation
- Conditional Formatting — KPI Visualization

## Key Findings
- Identified sales strategies projecting ~10% revenue improvement
- Reduced reporting effort by 30% through dynamic dashboards
- Discount and margin analysis revealed 5–8% profit improvement opportunity
